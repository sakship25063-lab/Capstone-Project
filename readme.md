Lost & Found Portal
The Lost & Found Portal is a simple web application that allows users to:

Add lost or found items
View all items
Search items
Delete items

The goal is to build a full-stack application using:

HTML, CSS, JavaScript (Frontend)
Flask (Backend)
In-memory list as database
